Grandma Please!

Video Game - Copy Editor

A visual novel video game utilizing Ren'Py with a fantastical original story about a druid traveling between worlds. The writer utilized a non-traditional script format to compensate for the non-linear story structure. Editing the script required learning an original script structure.
The project details can be found on the Gamespawn, game development club, website.

  • Copy edited new chapters of script on a weekly basis
  • Provided feedback aimed at assisting the author reach their intended vision for the project
  • Reviewed over 60,000 words of script
